# Dispatch game
an attempt at creating a web mission dispatch game using js
(inspiration: Adhoc-Dispatch)

## Character classes
- **Healer**: Divine/nature caster focused on restoration and protection
- **Mage**: Arcane scholar wielding destructive magic
- **Warrior**: Frontline fighter specializing in melee combat
- **Tank**: Heavily armored defensive specialist
- **Marksman**: Ranged specialist with keen accuracy
- **Rouge**: Master investigator, trap finder and information gatherer
- **Assassin**: Stealth expert dealing critical strikes

## Stats System:
#### Physical Stats:
- **Combat** - Weapon skill
- **Vigor** - HP/Endurance
- **Mobility** - Speed/Agility
- **Fortitude** - Defense/Armor

#### Mental/Social:
- **Intelligence** - Knowledge/Reasoning
- **Charisma** - Social/Leadership
- **Perception** - Awareness/Investigation

#### Magic Stats:
- **Arcane** - Offensive magic (Mage specialty)
- **Restoration** - Healing/Support magic (Healer specialty)
